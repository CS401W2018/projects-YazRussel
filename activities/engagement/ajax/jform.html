  1. What is AJAX?
    Asynchronous, means out of order. It allows web pages to update asynchronously by exchanging small amounts of data with the server behind the scenes. It is possible to update parts of a web page, without reloading the whole page, leading to a quicker interactive user experience.
  2. Why do you put your JS Code in a separate file?
    We need to separate JS into external files to enhance website organization and maintainability by allowing code reuse across multiple pages and simplifying updates.
