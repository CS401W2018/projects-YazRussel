1. Explain the difference between inline, embedded, and external CSS
CSS, Cascading Style Sheets
  • Inline CSS - Inside the tag CSS. Like style="", this is also known as HTML Attributes. Completely remove in HTML, some other browser can read it and some cannot.
  • Embedded CSS - have its own style tag. We have to use the CSS selector so that we can specify style we want to change.
  • External CSS - Just like embedded, but the style is in another file. If you are using inline and embedded in your html code, then it will override what you have designed in your external css.
2. Explain why you should only use external CSS
Seperating the css and html program to make your code clean and more manageable.
