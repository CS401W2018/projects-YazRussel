What is the difference between an ID and a Class?
ID should be unique, like you are naming someone. You can't reuse, however your code may run but there will be some error. Usually the ID is a noun. You cannot put two id in ID. Meanwhile, a class should be an adjective, and you can reuse it. For class, a tag can put more class in the selector area just put a space.
What is the difference between a SPAN tag and a DIV tag?
When you have to select certain thing. Usually it creates a box, but no one can see it. That everything that comes on top of it will go above it, and the one goes after it will move the below it. Meanwhile, the span is just selecting a group of text you want to emphasize.
What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
Div tag came first. Header, main, footer, is just underneath of DIV. DIV have no semantic meaning compared to the header, main, and footer.
