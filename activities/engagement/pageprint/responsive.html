  1. Why do we need a viewport meta tag?
    The viewport meta tag is really important because it helps control how a website looks on your phone's browser. It lets web developers set the size and shape of the part of the browser window where the website shows up. This means they can make sure the website looks good and works right on different-sized screens.
  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?
    The flex-direction: row; will take priority because it is defined last in the CSS. It override the previous one
  3. How do you change the CSS for a web page displayed on a phone?
    Use this "@media only screen and (max-width: 600px)"
