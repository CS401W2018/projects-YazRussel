Internet

The Internet allows computers and other devices to communicate, and send or receive data from each other. It is a useful network way to exchange information across the whole world.

Web

Web is also known as World Wide Web, it gives an easy access to information through the interlinked hypertext documents that are accessible via internet. It has browsers that allows the user to interact with its content.

How are the Internet and Web different?

Internet allows all devices to interact with each other, while web gives service to the users and providing them the content of the linked documents through internet.

ASCII and Unicode

ASCII or American Standard Code for Information Interchange is a common character encoding for text data and used to encode 128 specified characters, this was originally designed for use with teletypes.
Nowadays, ASCII is replaced by Unicode because of it is designed to support all of the world writing systems in the world that can now be digitalized.

What is UTF-8

UTF-8 is an econding standard used to display and use text for computers.

URL

URL or Uniform Resource Locator used to locate the address of an specific information to access its resources on the internet.

Domain Name

It is a name that you give to a website to make it easy to find it on internet instead of using the IP addresses.

HTTP

HTTP or Hypertext Transfer Protocol is a protocol used to facilitate the transfer the data between the web servers and the users. However, it is not secured as the HTTPS.

HTTPS

HTTPS or Hypertext Transfer Protocol Secure is another version of HTTP but more secured. It encrypts all of the sensitive data away from possible attack of cybercriminals.

IP Address

IP Address gives a unique identifier for every device who are connected to the internet. This allows all device to connect and communicate with other device.

DNS

DNS or Domain Name System helps translate the domain name into its corresponding IP address which allows to connect the browser to its server.

W3C Recommendations

W3C or Wold Wide Web Consortium is a standard guideline for web to ensure that all websites are accessible on devices for everyone.

XML, XHTML, HTML, HTML5

XML or Extensible Markup Language provides a structured data that can be shared to different systems in internet.

XHTML or Extensible Hypertext Markup Language it is another version of HTML that is XML-base.

HTML or Hypertext Markup Language a programming language used for creating web pages.

HTML5 is the latest version of HTML with few features for handling the content of the web.
